## 💡 Tips for Backend Integration

1. **Start with Authentication**
   - Test login/logout first
   - Ensure token storage working
   - Test role-based access

2. **One Module at a Time**
   - Start with User CRUD
   - Then Jurusan, Kelas, etc.
   - Test thoroughly before moving on

3. **File Upload Last**
   - Get basic CRUD working first
   - Then add file upload
   - Test with small files first

4. **Use Mock Data as Reference**
   - Mock data structure = expected API response
   - Field names match expected API fields
   - Relationships already defined

5. **Test Error Cases**
   - Invalid token
   - Network errors
   - Validation errors
   - File too large
   - Wrong file type
